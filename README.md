# GadsLeadershipBoard
Google Africa Developer Scholarship, Associate Android Developer Practice Project

# Approach To the Task
The Application makes use of 
- Multi-Activity Project
- View pager
- Customizing Toolbar
- RecyclerView
- Making Network Requests with Retrofit Library
